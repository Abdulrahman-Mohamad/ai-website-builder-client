# AI Website Builder — Client

A SaaS web application where users describe a website in plain text and AI generates it in real time. Users can preview, edit content inline, manage projects, and publish to the web.

## 🚀 Live Demo
[https://ai-website-builder-client-xi.vercel.app/](https://ai-website-builder-client-xi.vercel.app/)

> ⚠️ The backend runs on a free server — first response may take 30–60 seconds to wake up.

## ✨ Features
- AI-powered website generation from a text prompt
- Real-time step-by-step generation UI
- Inline content editor (click any element to edit text, font, alignment)
- Multiple version history per project
- Community page to explore websites built by other users
- Dashboard to manage all your projects
- Credits & pricing system with upgrade flows
- Authentication (login/signup)

## 🛠️ Tech Stack
- React 19, TypeScript, Vite 7
- Tailwind CSS v4, Framer Motion
- React Router v7
- Axios
- Better Auth (client)

## ⚙️ Getting Started

```env
VITE_API_URL=http://localhost:5000
```

```bash
npm install
npm run dev
```
